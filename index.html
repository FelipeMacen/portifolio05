Projeto Corrida

O projeto é um jogo criada a partir de JavaScript pelo site P5.js. O objetivo é fazer os jogadores se divertirem com um jogo de corrida.

O cenário do jogo é composto por um ambiente natural, com céu, sol, grama, nuvens e obstáculos visuais na forma de árvores, cujas posições são geradas aleatoriamente a cada execução. A interação ocorre por meio do evento keyReleased(), onde cada jogador avança uma distância aleatória ao pressionar sua respectiva tecla, adicionando um elemento de imprevisibilidade e diversão.

O sistema de verificação identifica automaticamente o vencedor, exibe uma mensagem na tela e encerra o jogo ao detectar que um dos jogadores cruzou a linha de chegada.

Este projeto destaca-se pelo uso de arrays para gerenciar múltiplos jogadores e objetos do cenário, aplicação de geração procedural para posicionamento dos obstáculos, e integração de elementos gráficos que enriquecem a experiência visual e interativa. É um excelente exemplo para demonstrar conceitos fundamentais de programação, como controle de eventos, manipulação de arrays, lógica condicional, e desenvolvimento de jogos 2D simples.

Projeto Monalisa

Este projeto consiste no desenvolvimento de uma aplicação interativa utilizando a biblioteca p5.js, onde é criado um personagem estilizado que responde dinamicamente à movimentação do cursor do usuário. O personagem é composto por formas geométricas básicas, como círculos, linhas e triângulos, que representam o rosto, os olhos, o nariz, a boca e as sobrancelhas.

A principal funcionalidade interativa do projeto é o acompanhamento do movimento do mouse pelos olhos do personagem. Isso é realizado através da função map(), que ajusta as coordenadas das pupilas conforme a posição do cursor, criando o efeito visual de que o personagem está seguindo o usuário com o olhar.

Além disso, o código inclui uma função opcional de depuração, que imprime no console as coordenadas do mouse quando o botão é pressionado. O fundo vibrante e a paleta de cores destacam o caráter lúdico e expressivo da interface.

O projeto demonstra conceitos fundamentais de programação gráfica, como manipulação de coordenadas, interatividade em tempo real, e construção de formas através de primitivas geométricas, sendo ideal para fins educacionais e criativos na introdução à programação visual.

Projeto Quente-frio

Este projeto consiste no desenvolvimento de um jogo interativo utilizando a biblioteca p5.js, cujo objetivo é localizar uma palavra escondida em uma posição aleatória da tela. O sistema utiliza como principal elemento de interação um círculo que segue o cursor do mouse, variando seu diâmetro de acordo com a proximidade da palavra oculta: quanto maior a distância entre o cursor e a posição da palavra, maior será o círculo; à medida que o usuário se aproxima, o círculo diminui de tamanho, funcionando como uma pista visual.

A posição da palavra é gerada aleatoriamente no início da execução, com leves variações dinâmicas para simular movimento e aumentar a dificuldade. Quando o usuário encontra a posição correta — indicada pela proximidade mínima entre o cursor e a palavra — uma mensagem de sucesso é exibida ("Encontrei!") e o programa é interrompido, sinalizando o fim da busca.

Este projeto demonstra habilidades essenciais de programação interativa, como geração de números aleatórios, manipulação de distâncias, feedback visual baseado em variáveis e controle de fluxo. Além disso, promove conceitos lúdicos e educacionais, reforçando a lógica computacional e a percepção espacial de maneira criativa.
